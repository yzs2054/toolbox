# 需求文档

## 项目概述

多功能工具箱，本地 Web 应用，浏览器操作。支持视频下载、音频提取等功能，后续持续迭代。

## 用户场景

- 从微信公众号、百度新闻等网页提取并下载视频
- 查看历史下载记录并重新获取已下载文件
- 音频提取与格式转换
- 其他日常媒体处理需求

## 功能需求

### 已完成

| 功能 | 说明 | 版本 |
|---|---|---|
| 视频下载 | 从网页提取视频信息并下载，支持微信公众号（腾讯视频）、mpvideo 直链、HTML5 video、直接 mp4/m3u8 链接 | v1.0.0 |
| 自动更新 | 检查 GitHub Releases 新版本，一键下载升级 | v1.0.0 |
| CI/CD | GitHub Actions 自动打包 Windows exe，打 tag 触发 | v1.0.0 |
| 百度新闻视频 | 支持 `mbd.baidu.com` 落地页，经好看视频侧拉取 360P/480P/720P/1080P 直链 | v1.1.0 |
| 下载记录持久化 | `downloads/history.json` 跨重启保留任务历史，含时间戳与输出文件名 | v1.1.0 |
| 记录文件直链 | 完成的任务在 UI 提供「下载文件」链接 | v1.1.0 |
| 端口可配置 | 通过 `PORT` 环境变量覆盖默认端口 8080 | v1.1.0 |

### 规划中

| 功能 | 说明 | 优先级 |
|---|---|---|
| 音频提取 | 从视频中提取音频，支持 mp3/wav/aac 格式 | 高 |
| 格式转换 | 音视频格式互转，如 mp4→avi, wav→mp3 | 高 |
| 视频号下载 | 微信视频号分享链接解析下载（依赖 Cookie） | 高 |
| 视频截取 | 截取视频片段，指定起止时间 | 中 |
| 图片提取 | 从网页批量提取图片 | 中 |
| 文本处理 | JSON/CSV 格式化、转换 | 低 |

## 非功能需求

- 单文件分发（exe + ffmpeg），无需安装 Python
- 启动后自动打开浏览器
- 跨平台支持（Windows 优先）
- 界面简洁，响应式适配

## 技术约束

- 后端：Python + Flask
- 前端：HTML + Tailwind CSS + 原生 JS
- 视频下载：yt-dlp（Python 库调用）
- 音视频处理：ffmpeg
- 打包：PyInstaller / Nuitka
- CI/CD：GitHub Actions
